🔥 ຕັ້ງຄ່າ Firebase

Firebase ຊ່ວຍໃຫ້ຂໍ້ມູນ sync ໂດຍອັດຕະໂນມັດລະຫວ່າງ Browser ແລະ ໂທລະສັບທຸກໂຕ.
ເຮັດຄັ້ງດຽວ ຂໍ້ມູນຈະຄົງຢູ່ຕະຫຼອດ.

🔧 ວິທີຮັບ Firebase Config (ຟຣີ):
  1. ໄປທີ່ console.firebase.google.com
  2. ກົດ "Add project" → ຕັ້ງຊື່ → ສ້າງ Project
  3. ໃນ Project → ກົດ "Web app" (</>) → ລົງທະບຽນ App
  4. Copy ຄ່າ firebaseConfig ມາໃສ່ຊ່ອງຂ້າງລຸ່ມ
  5. ໄປທີ່ Firestore Database → ສ້າງ Database → ເລືອກ "Start in test mode"

* ຖ້າບໍ່ຕ້ອງການ Firebase ກົດ "ຂ້າມ" ໂປຣແກຣມຈະໃຊ້ຂໍ້ມູນໃນ Browser ຢ່າງດຽວ
